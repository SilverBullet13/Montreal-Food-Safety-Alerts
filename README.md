## About The Project

This project uses public data from the City of Montreal
to track food establishments that have received violation
notices during health inspections and provides alerts 
and services based on that information.

## Getting Started

#### Prerequisites 
- python 3.12

#### Installation
- ```pip install -r requirements.txt```

#### Run
After installing the necessary requirements, run the following 
command to collect the data from the City of Montreal public 
database.
- ```python collect_data_script.py```

This command launches the app : 
- ```python app.py```
